Web Application Migration Automation

Using Github Actions

Summary

This is a webpage that will describe how you can migrate a web application using github actions from your github directly to AWS cloud. The migration happens only when a commit to the main branch happens.

Steps

  • 1. Create a new EC2 Instance

  • 2. Create Gibhub Action Secrets

  • 3. Create the Workflow

  • 4. Test the application

Github Action Secrets

Host Name

This is an action secret to identify which server the application will be migrated to. This is the Public DNS of the EC2 instance.

Username

This is ubuntu.

Key used to connect to the EC2 instance

While creating the EC2 instance, create a new key file as a .pem file. The content of the pem file is the EC2_SSH_KEY.
